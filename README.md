# C++ Fundamentals

This repository contains beginner-level C++ programs written while learning the core concepts of C++.

## Topics Covered

- Basic Output (cout, endl, \n)
- Input Handling (cin, getline)
- Data Types
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Arrays (1D and 2D)
- Strings
- Functions (void, parameterized, return)

## Folder Structure

01_basics → Basic output programs  
02_input_output → Input and output programs  
03_datatypes → Programs demonstrating data types  
04_conditions → If-else and switch programs  
05_loops → Loop programs  
06_arrays → Array programs  
07_strings → String programs  
08_functions → Function programs
